# Images Directory

This directory should contain the following taxi-related images:

## Required Images:
- taxi-hero.jpg (1920x1080) - Main hero image
- sedan.jpg (800x600) - Standard sedan vehicle
- executive.jpg (800x600) - Executive/luxury vehicle  
- minivan.jpg (800x600) - Minivan/group transport
- city-tour.jpg (800x600) - City tour service image
- airport-transfer.jpg (800x600) - Airport transfer service
- business-travel.jpg (800x600) - Business travel service

## Fallback System:
When these images are not present, the website automatically uses professional Unsplash photos as fallbacks.

## Image Requirements:
- Format: JPG/JPEG recommended
- Hero image: 1920x1080 minimum
- Service/fleet images: 800x600 minimum
- Professional quality taxi/transportation photos
- Optimized for web (under 500KB per image)
